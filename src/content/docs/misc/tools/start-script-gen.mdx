<details>
  <summary>我应该选择哪些 flags？</summary>
  如果您使用的是现代 JVM（例如 17、21 或更高版本），请首先尝试不使用任何标志。
  现代 JVM 非常擅长使用默认的垃圾回收设置来处理各种应用程序。

  如果您遇到垃圾回收问题，或者运行的是较旧版本的 Minecraft/Java，请尝试使用 [Aikar 的 flags](/paper/aikars-flags)，这些标志是专门为 Minecraft 优化的。

  如果您正在为 [Velocity](https://papermc.io/software/velocity) 生成启动脚本，请选择 “Velocity”。
</details>
