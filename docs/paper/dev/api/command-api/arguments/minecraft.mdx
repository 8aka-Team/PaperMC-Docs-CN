# Minecraft 参数
[参数和字面量](../basics/arguments-and-literals) 页面涵盖了最常用的原生 Brigadier 参数。但 Minecraft（和 Paper）定义了更多参数。这些参数可以通过
<Javadoc name='io.papermc.paper.command.brigadier.argument.ArgumentTypes'>`ArgumentTypes`</Javadoc> 类在静态上下文中访问。我们将在本节中介绍所有这些参数。
