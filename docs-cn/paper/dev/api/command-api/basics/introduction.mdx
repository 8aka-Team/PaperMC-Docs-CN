# Brigadier 命令简介
Paper 的命令系统是基于 Minecraft 的 Brigadier 命令系统构建的。该系统提供了一种强大而灵活的方式来定义命令和参数。
相比之前广泛使用的 Bukkit 命令系统，它具有许多优势：
- 开发者需要对参数进行的解析或错误检查更少。
- 通过客户端错误检查提供更好的用户体验。
- 与重载事件集成，允许定义数据包中可用的命令。
- 更容易创建子命令。

:::danger[实验性]

Paper 的命令系统仍处于实验阶段，未来可能会发生变化。

:::
