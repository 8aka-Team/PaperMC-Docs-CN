# 欢迎来到 Velocity 文档

Velocity 是一个高度可扩展、灵活的 Minecraft 代理服务器。

## 入门指南

开始使用 Velocity 非常简单。通过我们的[入门指南](admin/getting-started/README.mdx)开始使用。

## 我需要帮助

如果你需要 Velocity 的帮助，请在以下地方寻求帮助：

### 📖 这些文档

我们在新网站上投入了大量精力尽可能详细地记录 Velocity，我们的文档覆盖范围将继续扩大。我们强烈建议你查看文档侧边栏中的相关资源。使用这些文档中的资源自助可以为我们所有人节省时间。

我们建议你从[常见问题](admin/getting-started/faq.mdx)开始搜索。大多数关于 Velocity 的常见问题都在那里得到了解答。

如果我们回复你的问题时链接回这里，请不要觉得被冒犯。询问已经在文档中回答过的问题对任何人都没有帮助。

### 💬 我们的 Discord

如果你已经搜索过文档但没有找到问题的答案，那么是时候[加入我们的 Discord](https://discord.gg/papermc)来提问了。
