# 物品命令转换器

在 1.20.5 版本中，Mojang 已经从非结构化的 NBT 转移到了所谓的物品组件。虽然 Minecraft
会自动为您更新所有现有的物品，但只有 Paper 会在可能的情况下更新包含物品的命令和文本
组件。这个工具可以帮助您将旧的物品命令转换为新格式。

请注意，如果您想在 summon 命令之外转换实体数据，您需要选择 *Entity Argument* 模式，
因为我们需要实体类型来正确转换数据。
